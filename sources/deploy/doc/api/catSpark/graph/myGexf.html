case class myGexf(author: String, description: String, directed: Boolean) extends Product with Serializable

Implements useful methods to manage the graph file format gexf: https://gephi.org/gexf/format/ . Those method are based on the book: "Spark graphx in action" by Michael S. Malak and Robin East. . Graphs files generated can be visualized with Gephi: https://gephi.org

author

Person that creates the graph

description

Description of the content of the graph

directed

True for 'directed' graph or false for indirected
